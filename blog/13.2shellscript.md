## 利用test指令测试

* 通过$? 配合&& 和 || 可有做判断然后给出执行命令 ，test 命令也可以做很多判断 
```bash
lfg1000708009:/home/gaocong # test -e /tmp/abc && echo "exist" || echo "No exist"
exist
lfg1000708009:/home/gaocong # test -e /tmp/abc123 && echo "exist" || echo "No exist"
No exist
```

## 不定循环
### 格式
* 第一种: 当...时,条件成立时执行.不成立时结束
```bash
while [ condition ]  #条件判断式
do                   #循环开始
    程序段 
done                  #循坏结束
```
* 第二种: 条件成立是结束循环,不成立时就执行
```bash
until [ condition ]
do 
    程序段落
done
```
